# /todo-config — Проверить конфиг доски

Проверь наличие и корректность файла `.qwen/todo-ai.json` в корне текущего проекта (`WORKSPACE_PATH/.qwen/todo-ai.json`).

## Логика

### 1. Файл НЕ существует
Создай его с содержимым:
```json
{
  "board": "main"
}
```
Сообщи пользователю: «Конфиг создан с доской по умолчанию `main`.»

### 2. Файл существует
Попробуй прочитать и проверить:
- ✅ Валидный JSON
- ✅ Есть ключ `"board"`
- ✅ Значение `board` — непустая строка
- ✅ Имя содержит только буквы, цифры, дефис, подчёркивание

**Если всё ок** — покажи пользователю текущий конфиг.

**Если невалиден** — исправь и перезапиши, сообщи что было исправлено.

## Пример вывода

**Конфиг создан:**
```
Создан .qwen/todo-ai.json → {"board": "main"}
```

**Конфиг корректен:**
```
.qwen/todo-ai.json: {"board": "my-project"}
→ AI-агент работает с доской: my-project
```

**Конфиг исправлен:**
```
Конфиг был невалидный (отсутствовал ключ board). Исправлен → {"board": "main"}
```
