# /todo — Создать задачу

Создай новую задачу в колонке "Бэклог-AI" Kanban-доски todo-ai.

Используй MCP инструмент `create_task` с параметрами:
- `title` (обязательно) — заголовок задачи (до 100 символов)
- `description` (опционально) — описание задачи
- `context_ai` (опционально) — контекст для AI-агента
- `checklist` (опционально) — чеклист: `{"title": "...", "tasks": [...]}`

После создания покажи пользователю ID и название созданной задачи.
