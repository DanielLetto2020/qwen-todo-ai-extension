# /todo-next — Показать следующую задачу

Покажи следующую задачу из колонки "Сделать" Kanban-доски todo-ai.

Используй MCP инструмент `todo_next`. Задача НЕ меняет статус — только просмотр.

Покажи пользователю:
- Номер задачи (#id_task)
- Название
- Описание (если есть)
- Приоритет
