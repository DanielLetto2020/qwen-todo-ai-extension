# /todo-next-run — Взять задачу в работу

Возьми следующую задачу из колонки "Сделать" и перемести её в "В работе" (status=2 → status=3).

Используй MCP инструмент `todo_next_run`.

После выполнения покажи пользователю:
- Номер задачи (#id_task)
- Название
- Полный контекст: title, description, context_ai, checklist, ai_notepad
- Предложи выполнить задачу
