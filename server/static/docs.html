🚀 Автозапуск UI сервера

UI сервер запускается автоматически при каждой загрузке MCP сессии:

  1. 1. Проверяет — работает ли UI на порту 8167
  2. 2. Если нет — запускает как независимый процесс
  3. 3. UI живёт после выхода из сессии Qwen Code

Не нужно запускать вручную! Просто откройте http://localhost:8167 в браузере.

Ручной запуск (если нужен): cd ~/.qwen/extensions/todo-ai && python -m server.main

Свой порт: python -m server.main --port 9000

REST API

Задачи

GET /api/tasks?status=2 Список задач
GET /api/tasks/{id} Получить задачу
POST /api/tasks Создать задачу
PUT /api/tasks/{id} Обновить задачу
PATCH /api/tasks/{id}/status Изменить статус
DELETE /api/tasks/{id} Удалить задачу

Доска

GET Доска с колонками и задачами

Чеклист

Чеклист хранится как JSON-объект внутри задачи в поле checklist. Доступ через создание/обновление задачи.

{
  "checklist": {
    "title": "План работ",
    "tasks": [
      {
        "title": "Настроить БД",
        "done": false,
        "children": [
          {"title": "Создать таблицы", "done": true, "children": []}
        ]
      }
    ]
  }
}

Правила

1. Только creator=human может создавать задачи в status=0 (Бэклог)

2. Только creator=ai может создавать задачи в status=1 (Бэклог-AI)

3. Все изменения автоматически логируются в поле log

4. Чеклисты: макс. 5 уровней вложенности (depth 0–4)
