# /todo-list — Список задач

Покажи список задач Kanban-доски todo-ai.

По умолчанию: используй MCP инструмент `todo_list` — покажи задачи из "Бэклог-AI" (status=1).

Если пользователь указал статус — используй `list_tasks` с параметром `status`:
- 0 = Бэклог
- 1 = Бэклог-AI
- 2 = Сделать
- 3 = В работе
- 4 = Требует внимания
- 5 = Готово
- 6 = Архив

Покажи пользователю таблицу: #ID | Название | Статус | Приоритет
